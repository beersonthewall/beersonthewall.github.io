Projects

ellers_rs

An implementation of eller's maze generation algortihm in rust. The algorithm uses a small amount of space, even for very large mazes. Only requiring the current row and previous row to generate a new row.

Brick

A brief attempt at writing an x86-64 kernel. Boots up and can print to the screen via the VGA buffer. Named after what it will most likely do to any actual hardware it's run on.

Frisbee or not

A small flask web app which uses a fastai model to classify an image of a sport as either ultimate frisbee or not.
